# Neovim Configuration

This is my personal `nvim` config.

requirements:

- `lua` based configuration.
- `lsp` 
- `treesitter`
- `telescope`
- `harpoon`
- Rust, Python, HTML, JavaScript and TypeScript support.

non-requirements:

- tabs or buffer line
- tree view or file exlorer
- VSCode-like UI.
